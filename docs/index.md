# Woda Prawo 2026

Krótka strona pośrednia dla publikacji aktu prawa — link do pliku PDF.

## Jak to działa

Ta strona jest przeznaczona do publikacji przez GitHub Pages. Po umieszczeniu pliku PDF w katalogu `docs/` i włączeniu Pages (gałąź główna, folder `/docs`) strona będzie dostępna publicznie pod adresem:

https://thajka.github.io/woda-prawo-2026/

## Pobierz akt (PDF)

[Pobierz PDF aktu](akt.pdf)

---

Instrukcja Bieżąca

1. Umieść plik PDF pod `docs/akt.pdf` w tej samej gałęzi.
2. Przejdź do Settings -> Pages w repozytorium.
3. Wybierz publikację z: Branch = główna (np. "main"), Folder = "/docs".
4. Zapisz i poczekaj kilka minut na publikację — GitHub wyświetli krótką publiczną nazwę strony.

Jeśli chcesz, mogę również dodać przykładowy plik PDF (placeholder) do `docs/akt.pdf` teraz, albo utworzyć stronę jako `docs/index.html` zamiast Markdown — co wolisz?
